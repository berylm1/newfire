## Architecture Decisions

1. **Cloudflare Tunnel for production ingress** (not Tailscale Funnel or direct zrok2)
   - Proper TLS with correct domain certificate
   - DDoS protection from Cloudflare
   - No public ports needed, tunnel is outbound-only
   - Custom domain works properly

2. **Keep zrok2 for self-hosted zero-trust layer**
   - Own the entire stack, no third-party dependency for core networking
   - OpenZiti provides identity-based access control
   - Production-ready once Caddy TLS overlay is added

3. **Tailscale Funnel as dev/demo access**
   - Quick access for internal testing
   - Does not replace production domain

4. **Hybrid model routing**
   - Local GPU models for 90% of tasks (free after hardware cost)
   - Cloud models (OpenRouter) only for tasks needing frontier intelligence
   - Budget capped at $0.50/agent/day
